CODING BRICKS IN DETAIL 🔎

I know what you're thinking... Whaaat?! 10 programming languages in one course? 30 apps?

Yep, you're right, 10 programming languages, 30 apps.


The focus of this course is to teach you how to code, not how to master a language.

Instead you're going to learn all about variables, scopes, methods and functions, parameters/arguments, classes, loops, objects, data types and more!


You will learn how to think like a programmer, how to code real apps and most important, you will become a programmer with solid understanding of what's going on under the hood.


I've been in your place, and the reality is, it's hard to learn how to code, and how to do it well.

The programming world is full of languages, tools, frameworks and libraries, and it's difficult to find where to start.


That's why I created this course, to teach people like you how to code from the very beginning, from A to Z. Stop fidgeting through the noise, I'll teach you everything from HTML and CSS up to how to configure servers and deployment.


Are you ready to become a programmer?

COURCE CONTENT 💻

1. Introduction 👨‍💻

In this first module, you'll learn how this course is structured and you will install the necessary tools we'll be using like a code editor and a terminal.


Also, you'll get an invite to the Coding Bricks Slack group for you to meet other awesome students and collaborate on projects, chat about anything, ask questions and help other peers along the journey 😃

2. Learning C 💻

The C programming language is the most important language of all times! In this module you'll learn the basics of programming by building 3 apps written in C 😉


You'll also learn some quick history of programming, use cases and how to use the terminal and move into directories.

3. Learning C++ 🖥️

C++ is a programming language influenced by C, which brings to the table Object-Oriented Programming (OOP) features 😲


You'll learn the differences between an Object-Oriented programming language (C++) and a structured language (C) in this module.

4. Learning Java ☕

Java is one of the most used programming languages in the world 🌎 it's almost everywhere, learn it by coding 3 apps and deeply understand Object Oriented Programming (OOP).


At this point you'll know how to work with classes and objects, understand inheritance, polymorphism and other pillars of OOP.

5. Learning Go 🐿️

Go (or golang) is a programming language created at Google, it's statically typed, really fast and very fun to learn!


Learn this programming language by building 3 useful applications, including a to-do list app ✅ to become more productive!

6. Learning Python 🐍

Python is a very powerful programming language used by companies like Disqus, Bitbucket, Quora, Instagram, Pinterest and NASA!


You'll learn how to code in Python by building 4 cool apps, including a fun to play video-game! 👾


After finishing this module you'll get the confidence of writting powerful programs and get ready to learn more advanced techniques!

7. Version Control 🐙

Version Control is essential for every developer, it allows you to keep track of your code changes over time, save your project progress, contribute to Open Source projects and more.


In this module you'll learn how to work with Git and GitHub, pull and push changes to repositories and how to contribute to Open Source projects 🙌

8. Web Development 🕸️

In this module you'll learn how the Web works and communicates, you'll also build your first website with HTML and CSS, deploy it to different servers and get your perfect domain name.


Learn Responsive Design, Flexbox, CSS Grid, PostCSS, CSS Next, CSS Animations, Bootstrap, Foundation, Bulma, Tailwind CSS, Sass, Less, Stylus, Gulp, Grunt and many many more tools! ⚙️

9. Learning JavaScript 💛

JavaScript is the programming language of the Web, and it's used by almost every site in the world, if you're a web developer you must learn it.


Learn JavaScript by making 4 really cool apps that interact with API's, including Twilio to send SMS! 😄


You're going to use jQuery, API's, NodeJS, Pug, Handlebars, Express, Webpack, Browserify, ES6, NPM, Yarn and more!

10. Debugging 🐛

Debugging is the art of finding bugs in your code 🔎 and fixing the errors.


Learn debugging techniques and how to examine your code, find errors quickly and discover what different types of errors/warnings there are, and the most common ones.


Catch those pesky bugs in your code with ease and become a bug master.

11. Learning SQL 🗄️

SQL stands for Structured Query Language, and it's a language designed to read and write into databases.


Learn SQL by creating and managing various databases using MySQL 🐬 an open-source relational database management system.


Learn how to store and retrieve information from DB's with this querying language used by large companies across the world.

12. Learning PHP 🐘

Learn PHP and how a server-side programming language works and start building rocking web applications! 🎸


You'll not only learn how to program in PHP, but you'll get to know how a backend system works under the hood.


Build 3 web applications that interact with MySQL databases, and get ready to make dynamic websites!

13. Learning Ruby 💎

Ruby is a dynamic, open source programming language with a focus on simplicity and productivity.


The syntax of Ruby is elegant, natural to read and easy to write, simple in appearance but very complex inside.


Learn how code in Ruby by building 3 incredible applications, and have fun with this beautiful programming language! 💖

14. Searching for Answers 🤔

Every developer struggle with one thing or another, whether it's a bug or some implementation, programmers need to be good at learning how to search for answers 🕵️


Searching for answers and learning how to do it well is a skill every developer in the world must master at some point in life, learn how to do it now and become a problem solver.

15. Learning Bash 💲

Bash is a Unix shell and command language, and it's super useful! 👌 Bash allows you to write shell scripts that interact with Unix systems.


If you use MacOS or Linux, or deploy your apps to Linux servers, you may want to learn Bash to create handy scripts that allows you to have better control over your system.

16. Gluing things together 😎

In this module you're going to combine different tools and languages to build a complex application! 🚀


A lot of the times, new developers get confused on how to work with different technologies to accomplish X or Y.


Learn how to work with multiple languages at the same time and get a better perspective of the programming ecosystem.

17. Conclusion and Inspiration ⭐

From now on, you can call yourself a programmer! 👏 Congratulations! 🎉


This module is an overview of what you've learned and what you can do with the knowledge acquired in this course.


You'll now be able to code apps and advance your career as a developer, with clear understanding of how to build awesome things!

18. Programming Communities 🌎

One of the best things to do in order to become a better developer is to get involved in a community of like-minded programmers 👨‍👩‍


Join Slack groups, forums, and follow top coders on Twitter and Facebook to stay up to date in the fast-pacing ever-changing amazing world of programming.

19. Surpriseee! 🎁

I have a special surprise for you! 🌟 this extra section is dedicated for YOU as a thank you for taking the course ❤️


In here you can find exclusive and awesome discounts on really useful things you may need in your developer journey 🙂.


Oh, and rad stickers for your computer, just like the cool developers 😝.

20. Where to Go from Here? 😄

"Move fast and break things. Unless you are breaking stuff, you are not moving fast enough." - Mark Zuckerberg


Go change the world, build awesome things, learn, practice, enjoy! 😃
